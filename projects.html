Bus Tracker

This code provides information for MBTA bus data to determine stops between Harvard and MIT. This code originate from an assignment in Week #8 of MIT's MERN Coding Bootcamp. The challenge was to animate and add features to a map view of a Boston bus route. Then, you need to and add an animated marker on the map to highlight the bus routes.
